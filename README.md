# DataX-Labs-Task-2

## Interactive Power BI Dashboard for Superstore Sales Analysis

This project is part of the DataX Labs Internship Task 2. It presents an interactive Power BI dashboard created using the Superstore dataset to analyze sales performance and generate business insights.

### Dashboard Features
- Sales by Category
- Sales by Region
- Profit by Sub-Category
- Sales by Segment
- Monthly Sales Trend
- KPI Cards (Total Sales, Total Profit, and Total Orders)

### Tools Used
- Power BI
- CSV Dataset
